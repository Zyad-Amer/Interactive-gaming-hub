# Interactive Gaming Hub for Children  

This project is an **Interactive Gaming Hub** built on the **STM32 microcontroller**

It contains **four classic games**:  
- **Snake**  
- **XO (Tic-Tac-Toe)**  
- **Memory Map**  
- **Quiz**
